Planning

    Accessibility 1

    I purposefully positioned a user-friendly navigation bar at the top of the page to make this website genuinely accessible. This deliberate design decision is a key component that improves the user experience as a whole. The site's navigation bar offers users an easy-to-use and transparent way to navigate between its many parts. Whenever I want to go from sharing my experience developing games to going into the intricate planning that goes into each one, all it takes is a click on the relevant navigation button. In addition to satisfying the demands of a wide range of users, this well-considered strategy makes navigation easy and efficient, reducing any issues a user may run into when navigating between the different areas.

    Accessibility 2

    Furthermore, the upper navigation bar enhances the site's accessibility while also adding a sophisticated touch to the overall layout. Its placement at the top of the page creates a visual hierarchy that makes it simple for people to browse and rapidly understand the content's structure. This design choice guarantees that users of all technical skill levels may easily browse the website and complies with current online accessibility standards. Through the integration of this user-focused navigation tool, my goal is to provide an inclusive online space where visitors can easily navigate my portfolio of game development work.

    Accessibility 3

    JavaScript was thoughtfully used into this portfolio's development to improve the navigation bar's functionality and provide users a smooth, intuitive experience. Using JavaScript and smooth scrolling functionality, the main goal was to make navigation easy for users so they could find and access specific sections of the portfolio quickly. Users could navigate between sections by simply clicking on the corresponding navigation link. This dynamic element made the website more engaging and made it easier for visitors to navigate around it overall by removing the need for them to manually scroll through lengthy pages.

    JavaScript was also used to dynamically highlight the navigation bar's active portion. The JavaScript code keeps track of the user's location on the page as they scroll through the content and dynamically refreshes the navigation bar to show which portion is now accessible. Users can easily discern their place in the portfolio thanks to this visual indication, which facilitates navigation and makes browsing easier. This portfolio presents a clean, modern appearance while also prioritising user ease and accessibility, in line with the ideals of current web design, thanks to the deft integration of JavaScript into the navigation mechanism.

Evaluation

what worked well

Implementation of Smooth Scrolling: The effective integration of JavaScript has allowed for smooth scrolling. By enabling a smooth transition between parts when a user clicks on a navigation link, it enhances the user experience overall. The simplicity and visual attractiveness of this code's navigation contribute to its efficacy.

Dynamic Navigation Highlighting: Another effective feature is the navigation bar's dynamic highlight of the current segment. The current segment of the material is dynamically updated in the navigation bar as the user scrolls through it. This visual signal enhances user engagement overall, facilitates navigation, and makes browsing easier.

Useful Navigation Buttons: Users are efficiently guided to particular portions of the portfolio by the navigation buttons. It is simple for users to locate pertinent information because each button is associated with a different component of the website. This simple navigation system improves the website's overall usability and conforms to user expectations.

what could be improved

area-specific pages: The content is arranged on a single page in the present design, but a more comprehensive portfolio needs a page for each area. A presentation that is better organised and targeted will be made possible by having separate pages for each project, background, planning, and other information. This will make the material more visible and make navigating easier.

Increased material: The portfolio will be much better if the material in each part is increased. Including more projects, thorough descriptions, and more extensive information will provide visitors with a complete picture of their expertise and experience. As part of this expansion, a wider range of initiatives and achievements would be highlighted in order to give a more thorough picture of capabilities.

Though the existing design includes responsive features, more improvements might be taken into account to support a wider range of screen sizes and devices. It is important to provide an excellent and consistent user experience across several devices. If there was more time, it would be carefully examined across a range of platforms and screen sizes, and adjusted to guarantee flawless operation and display.
